Student Course & Marks Analysis::

Project Overview:
This project demonstrates basic to intermediate SQL operations using a structured student dataset. It includes table creation, data insertion, and various queries to analyze student details, courses, and marks.

Database Structure:
The project/Analysis consists of three tables:

Students → Stores student details
Courses → Stores course information
Marks → Stores marks scored by students (with foreign key relationships)

Technologies Used:
MYSQL Woekbench
SQL (Structured Query Language)

Key Operations:
🔹 Level 1: Basic Queries
Display all student records
Select specific columns
Filter data based on conditions
🔹 Level 2: Filtering
Apply conditions using WHERE
Use operators like BETWEEN, !=, >
🔹 Level 3: Sorting
Sort data using ORDER BY
Ascending and descending order
🔹 Level 4: Aggregations
Count total records
Calculate average, maximum, minimum marks
Compute total marks

Learning Outcomes:
-Understanding relational database design
-Writing SQL queries for data analysis
-Using filtering, sorting, and aggregation
-Using primary and foreign keys
